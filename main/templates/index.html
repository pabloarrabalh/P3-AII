Bienvenido al Sistema de Recomendación de Películas

Este sistema permite realizar las siguientes operaciones:

Gestión de Datos

  • Cargar Base de Datos: Carga los datos del dataset en la base de datos
  • Cargar RecSys: Carga el sistema de recomendación en memoria

Consultas

  • Películas por Actor: Muestra todas las películas de un actor
  • Usuarios Más Activos: Muestra los 5 usuarios que más películas han puntuado
  • Recomendar Películas: Recomienda películas a un usuario

Nota: Antes de realizar consultas, asegúrese de haber cargado la base de datos y el sistema de recomendación.
